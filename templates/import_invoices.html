I clienti non presenti vengono creati automaticamente.
I PDF caricati vengono allegati alla fattura corrispondente e inviati automaticamente con i solleciti.

Note sul formato
  • CSV: separatore ; o , (rilevato automaticamente)
  • Date: GG/MM/AAAA oppure AAAA-MM-GG
  • Importo: 1.500,00 oppure 1500.00 (entrambi accettati)
  • Intestazioni case-insensitive (Nome = nome = NOME)
  • Excel: prima riga = intestazioni, le righe successive = dati
Da dove scaricare gli XML FatturaPA

Il formato XML è esportabile da quasi tutti i gestionali italiani:

  • Agenzia delle Entrate → "Fatture e Corrispettivi" → Cassetto fiscale → scarica ZIP
  • Fatture in Cloud → Fatture → Esporta → "XML SDI"
  • Aruba Fatturazione → Fatture emesse → Esporta → ZIP
  • Zucchetti / TeamSystem → Stampa e Esporta → "Tracciato XML"
  • Register.it → Archivio → Download fatture XML

💡 Carica anche un intero ZIP mensile: l'app processa tutti i file XML/p7m al suo interno.

Import da PDF

L'app prova a riconoscere automaticamente:

  • Numero fattura (parole chiave: "Fattura n.", "N° fattura"…)
  • Importo totale (parole chiave: "Totale", "Importo totale"…)
  • Date: emissione e scadenza
  • Cliente (dopo "Spett.le", "Cliente:", "Destinatario")
  • P.IVA del cliente

⚠️ I PDF scansionati come immagine non sono leggibili (servirebbe un OCR). I dati mancanti vanno completati manualmente dopo l'import.
